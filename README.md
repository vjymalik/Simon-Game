Game Overview: <br>
The Simon Game challenges players to test and improve their memory by following a sequence of colored button flashes. Each round adds a new color to the sequence â€” and the player must repeat it correctly to move on.<br>
<b>Gameplay flow:</b>
1. Press any key to begin.
<br>
1. Watch the sequence of flashes carefully.
<br>
1. Click the buttons in the same order.
<br>
1. Each level adds one more color to the sequence.
<br>
1. If you press the wrong button, the game ends and restarts.

Game Overview: <br>
The Simon Game challenges players to test and improve their memory by following a sequence of colored button flashes. Each round adds a new color to the sequence â€” and the player must repeat it correctly to move on.
<br>
<b>Gameplay flow:</b>
<br>
1. Press any key to begin.
<br>
2. Watch the sequence of flashes carefully.
<br>
3. Click the buttons in the same order.
<br>
4. Each level adds one more color to the sequence.
<br>
5. If you press the wrong button, the game ends and restarts.
